# Font Assets

This directory should contain the following font files:

## Required Fonts:
- `arcade.ttf` - Retro arcade font for UI elements

## Font Specifications:
- Format: TTF or OTF
- License: Ensure fonts are properly licensed for distribution
- Style: Retro/arcade style preferred

## Fallback:
If font files are not available, the game will use pygame's default font.

## Usage:
The game automatically loads these fonts when available, or uses system defaults if not found.
